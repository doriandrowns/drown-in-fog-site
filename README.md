# drown in fog site

## run locally

```bash
npm install
npm run dev
```

## build

```bash
npm run build
```

## github + vercel

1. create a github repo
2. upload every file in this folder
3. log into vercel with github
4. import the repo
5. click deploy
6. add your domain in vercel settings -> domains
